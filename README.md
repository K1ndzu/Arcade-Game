# Arcade Game

Criação de um jogo arcade com diversos mini jogos da época do nintendo, super nintendo e mega drive (1990 - 2000)

<img src="ImageReadMe\Lobby.png" height = 250>

A ideia é ter um lobby, como se fosse uma lan house aonde você tera acesso a toda essa variedade

<img src = "ImageReadMe\Space game.png" height = 250>

O primeiro jogo feito, foi inspirado nos spacing shooters, como aero fighters.

# Metas
* Conseguir um jogo responsivo, com controles que se adaptem ao estilo de jogo
* Conseguir desconstruir e reconstruir titulos grandes do genêro da epoca como Tetris, Mario, Kirby etc.
